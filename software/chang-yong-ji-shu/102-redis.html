# Redis

Redis 是一个开源的使用 ANSI C 语言编写、遵守 BSD 协议、支持网络、可基于内存亦可持久化的日志型、Key-Value 数据库，并提供多种语言的 API。

# 安装

采用 Docker安装Redis进行测试和学习，执行以下命令进行安装：docker run -d -p 6379:6379 --name redis redis。
