# 树莓派系统安装

[[toc]]

## 1. 下载系统

[https://shumeipai.nxez.com/download#os](https://shumeipai.nxez.com/download#os)点击选择`Raspberry Pi OS`进行下载系统，可以选择适合自己的版本，不同的系统配置会有所不同，目前只尝试了`Raspberry Pi OS`。
![树莓派](https://files.catbox.moe/tgrz2o.png)

## 2. 系统烧写

准备一个16G内存卡，安装之后系统大约4G，剩余的可以存储一些东西。推荐下载[etcher](https://www.balena.io/etcher/)
![etcher](https://files.catbox.moe/86a1ey.png)

选择镜像，点击制作即可
![制作](https://files.catbox.moe/q7zpfq.png)

## 3. 配置 wifi

创建`wpa_supplication.conf`文件，文件内容如下：

```conf
country=CN
ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
update_config=1
network={
    ssid="wifi名称"
    psk="wifi密码"
    priority=1
}
```

priority是优先级，network可以是多个，会先连接priority大的wifi

## 4. 开启 SSH

在boot盘的根目录创建`ssh`文件，无需任何后缀
