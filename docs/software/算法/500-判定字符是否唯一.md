# 判定字符是否唯一

## 题目内容

实现一个算法，确定一个字符串 s 的所有字符是否全都不同。

示例 1：

> 输入: s = "leetcode"
> 输出: false

示例 2：

> 输入: s = "abc"
> 输出: true

限制：

> 0 <= len(s) <= 100
> 如果你不使用额外的数据结构，会很加分。

## 本人理解

- 通过排序，比较上一个字符和这个字符相等即返回 false，不相等则继续比较。

```java
public boolean isUnique(String astr) {
    // 获得字节
        byte[] bytes = astr.getBytes();
        // 排序
        Arrays.sort(bytes);
        // 遍历
        for (int i = bytes.length - 1; i >= 1; i--) {
            // 判断是否相等
            if(bytes[i] == bytes[i-1]){
                return false;
            }
        }
        return true;

    }
```
