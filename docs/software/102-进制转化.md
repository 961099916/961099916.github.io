# 进制转化

进制之间的转化可以用于进行加密解密，可以自定义进制。

```java
public class Test {

    public static void main(String[] args) {
        System.out.println(toHex("155", NUM_CODE));
    }

    private static final String NUM_CODE = "0123456789ABCDEF";

    /**
     * num转化为16进制字符串
     *
     * @param num
     *            数字字符
     * @param code
     *            十六进制的编码
     * @return 十六进制的字符串
     */
    public static String toHex(String num, String code) {
        // 1. num转成数字
        Integer willHex = new Integer(num);
        // 2. 调用递归知道商为零
        String hexData = divUntilZero(willHex, 16, code);
        // 3. 反转
        StringBuilder stringBuilder = new StringBuilder(hexData);
        return "0x" + stringBuilder.reverse().toString();
    }

    /**
     * data除进制，直到商等于0
     *
     * @param data
     *            需转化的数据
     * @param code
     *            编码
     * @param hex
     *            进制
     * @return 进制字符串
     */
    private static String divUntilZero(Integer data, int hex, String code) {
        if (data == 0) {
            return "";
        }
        return code.charAt(data % hex) + divUntilZero(data / hex, hex, code);
    }
}

```
