# 数据看板

## 简介

每个完整的系统都需要数据的展示，而数据的展示随着项目的不断变大，数据展示的东西就越多，而随着微服务和项目的增多，数据展示越来越重要和支持多个项目的数据显示。而最近项目恰好是公司的数据看板，对该系统进行简单的分析。

## 系统基本原理

系统解决的基本问题：

- 如何能够支持多个项目的数据展示
- 如何配置实现想要观察的数据指标

### 1. 多个项目数据的展示

SpringBoot 虽然可配置多个数据源，但是却需要进行配置到项目中，而且如果新添加项目数据还需要进行修改。这种方式很不方便，SpringBoot 有`JdbcTemplate`可通过配置进行动态读取数据，从实现通过 SQL 进行动态查询相应数据库的数据。也可通过 API 获取数据，或者静态数据。

```java
private static List<Map<String, Object>> fetchSQL(MetricConfig config, Date tstart, Date tend, Long guid) {
      String jdbcSql = config.getJdbcSql();
      // JDBC模板依赖于连接池来获得数据的连接，所以必须先要构造连接池
      DriverManagerDataSource dataSource = new DriverManagerDataSource();
      dataSource.setDriverClassName("com.mysql.jdbc.Driver");
      dataSource.setUrl(config.getJdbcURL());
      dataSource.setUsername(config.getJdbcUserName());
      dataSource.setPassword(config.getJdbcPassword());
      // 创建JDBC模板
      JdbcTemplate jdbcTemplate = new JdbcTemplate();
      // 这里也可以使用构造方法
      jdbcTemplate.setDataSource(dataSource);
      // 替换sql语句里的时间变量 ${tstart}  ${tend} 
      Matcher matcher = SQLVarPattern.matcher(jdbcSql);
      List<Object> objs = new ArrayList<>();
      while (matcher.find()) {
          String trim = matcher.group(1).trim();
          if ("tstart".equals(trim)) {
              objs.add(tstart);
          } else if ("tend".equals(trim)) {
              objs.add(tend);
          } else if ("guid".equals(trim)) {
              objs.add(guid);
          }
      }
      jdbcSql = jdbcSql.replaceAll("\\$\\{(tstart|tend|guid)\\}", "?");
      // 执行sql
      if (objs.size() == 0) {
          List<Map<String, Object>> num;
          try {
              log.info("jdbc的ID" + config.getJdbcId());
              log.info("数据库查询SQL为>>>>>>" + jdbcSql);
              num = jdbcTemplate.queryForList(jdbcSql);
          } catch (Exception e) {
              return new ArrayList<Map<String, Object>>();
          }
          return num;
      } else {
          log.info("数据库查询SQL为>>>>>>" + jdbcSql);
          for (Object object : objs) {
              log.info("object为>>>>>>" + object);
          }
          List<Map<String, Object>> num = jdbcTemplate.queryForList(jdbcSql, objs.toArray(new Object[objs.size()]));
          return num;
      }
  }
```

### 2. 如何配置实现想要观察的数据指标

可以把页面分模块，每个模块有有多个指标，指标可通过指定相对应的方法获取相关的数据，然后通过前端设置绑定的 key 来获取对应的值，从而实现数据的获取，返回前端，前端通过判断类型从而实现指标的配置。需要前端写好各种数据指标的样式，然后根据 type 自动调用对应的指标样式
