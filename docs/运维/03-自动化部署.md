# 自动化部署（Docker+gitLab+Harbor+Jenkins）

## 环境准备

自动化部署的流程为代码提交到 git 库，然后出发 webhook，jenkins 自动拉取代码分支，进行打包镜像，发布到服务器中。本人环境已搭建，如需查看搭建方式，请查看上面几篇内容。需要安装的环境:

- Docker：进行容器化安装其他环境，部署是采用镜像部署
- GitLab：采用容器化部署，搭建私人代码库
- Jenkins：采用 Docker 容器化部署，通过私人代码库进行触发 webhook 进行执行脚本
- harbor：私人镜像仓库，自动化部署，把代码打包成镜像上传至私人镜像库，然后部署的服务器拉取镜像进行运行部署。
